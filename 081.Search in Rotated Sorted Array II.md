# Search in Rotated Sorted Array II

```java
public class Solution {
    public boolean search(int[] nums, int target) {
        int i = 0;
        while(i < nums.length && nums[i] > target){
            i ++;
        }
        while(i < nums.length && nums[i] < target){
            i ++;
        }
        if(i < nums.length && nums[i] == target){
            return true;
        }
        else{
            return false;
        }
    }
}
```
