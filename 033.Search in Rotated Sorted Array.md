# Search in Rotated Sorted Array

```java
public class Solution {
    public int search(int[] nums, int target) {
        int i = 0;
        while(i < nums.length && nums[i] > target){
            i ++;
        }
        while(i < nums.length && nums[i] < target){
            i ++;
        }
        if(i < nums.length && nums[i] == target){
            return i;
        }
        else{
            return -1;
        }
    }
}
```

Time complexity = n
