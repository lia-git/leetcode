# Move Zeroes (easy)
```java
public class Solution {
    public void moveZeroes(int[] nums) {
        int news = 0;
        for (int i = 0; i < nums.length; ++i) {
            if (nums[i] != 0) {
                nums[news] = nums[i];
                news ++;
            }
        }
        for (;news < nums.length; news ++) {
            nums[news] = 0;
        }
    }
}
```
