# Find Minimum in Rotated Sorted Array

```java
public class Solution {
    public int findMin(int[] nums) {
        int i = 0;
        int ans = nums[0];
        for(; i < nums.length - 1; i++){
            if(nums[i + 1] < nums[i]){
                ans = nums[i + 1];
            }
        }
        return ans;
    }
}
```
